About Steggy

Steggy is a webservice that gives users the ability to hide data in a cover file via the steghide library. The service focuses on simplicity to allow users to quickly hide and extract their data when necessary.

Steggy puts emphasis on anonymity, so users will not have to register or login when ever they want to take advantage of steggy's services.

Steggy also reveals all functionality through an API that users use as they wish.
