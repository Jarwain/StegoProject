API

Steggy API

Steggy provides a consumable API that allows users to implement Steggy's services in through their own web services.

The API provides an interface between the C++ Steghide Libary and the webservice. The API's two most important functions are the embed and extract function while retrieve and info are helper functions. JSON objects are used to send the fields designated in the table below.

URL GET POST DELETE
http://jarwa.in:8080/steganography/embed/

Accepts: {cover}, {embed}, [filetype], [password]

Returns: {id}, {checksum_sha1}
http://jarwa.in:8080/steganography/retrieve/{id}

Returns: File of {id}

Deletes: File of {id}
http://jarwa.in:8080/steganography/extract/

Accepts: {embed}, [filetype], [password]

Returns: Embedded file as blob
http://jarwa.in:8080/steganography/info/

Accepts: {cover}

Returns: {message} with details such as image capacity

Notes: The cover file field can only be a .jpg .jpeg .bmp .au .wav file type. The embed field can be any file type.
